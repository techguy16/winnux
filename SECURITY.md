# Security Policy

## Supported Versions

1.2.x will be supported until Winnux 2.0.x is released

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x (Alpha) | 🏭 In Development |
| 1.3.x (Beta)   | 🏭 In Development - 🛡️ Long Term Support |
| 1.2.x   | :white_check_mark: |
| 1.1.x   | :x:                |
| 1.0.x   | :x: |
| < 1.0   | :x:                |
